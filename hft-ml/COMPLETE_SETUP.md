## 🔧 TROUBLESHOOTING

### **Problem**: "market" not recognized

**Solution 1:** Open new Command Prompt (PATH updated)

**Solution 2:** Run directly
```bash
V:\quant_project\hft-ml\market.bat
```

**Solution 3:** Re-add to PATH
```bash
setx PATH "%PATH%;V:\quant_project\hft-ml"
```

### **Problem**: Can't login

**Default credentials:**
- Username: `admin`
- Password: `admin123`

**Create new account:**
- Select option 2 on first screen

### **Problem**: Password visible

**Password is automatically masked with `*`**
- Uses Rich library's password mode
- If still visible, update rich: `pip install --upgrade rich`
